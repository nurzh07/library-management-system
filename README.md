  ## Жобаның мақсаты:
Бұл жоба кітапхана жүйесін автоматтандыруға арналған веб-қосымша болып табылады. Қолданушылар кітаптарды іздеу, қосу, шығару және қайтару мүмкіндігіне ие болады. Сонымен қатар, кітапхана әкімшілері кітаптарды қосып, өзгертеді және өшіреді.

## Технологиялар:
- **Frontend**: React
- **Backend**: Node.js, Express
- **Database**: PostgreSQL / MongoDB
